## Player Attributes

Each player has attributes that influence their effectiveness:

| Attribute | Affects |
|-----------|---------|
| **Speed** | Movement efficiency, pursuit/escape ability |
| **Strength** | Physical engagements, blocking power |
| **Football IQ** | Decision quality, recovery ability |
| **Hands** | Catch success, interception ability |
| **Accuracy** | Pass precision (QB only) |

Higher attribute values generally improve performance in related actions. The game engine determines exact outcomes.

## Position Role Summaries

### Quarterback (QB)
- Field general who initiates plays
- Has a pass-target pattern defining throwable locations
- Key attributes: Accuracy, Football IQ

### Running Back (RB)
- Versatile offensive weapon
- Receives short passes, carries ball
- Key attributes: Speed, Strength, Hands

### Wide Receivers (WR1, WR2)
- Deep threats who run routes
- Create separation from defenders
- Key attributes: Speed, Hands

### Offensive Linemen (GL, GR, C_O)
- Protect QB through blocking
- Movement restricted to near line of scrimmage
- Key attributes: Strength, Football IQ

### Linebacker (LB)
- Most versatile defensive player
- Covers short routes, rushes QB, stops runs
- Key attributes: Speed, Strength, Football IQ

### Safety (S)
- Last line of defense, deep coverage
- Primary interception threat
- Key attributes: Speed, Hands, Football IQ

### Cornerbacks (CB1, CB2)
- Shadow receivers in coverage
- Match up against WR1/WR2 respectively
- Key attributes: Speed, Hands

### Defensive Linemen (TL, TR, C_D)
- Disrupt offensive plays at the line
- Rush QB, block passing lanes
- Movement restricted to near line of scrimmage
- Key attributes: Strength, Speed

## Strategic Considerations

### Matchup Analysis
- Compare Speed when evaluating pursuit/escape
- Compare Strength when evaluating physical contests
- Compare Hands when evaluating catch opportunities

### Attribute-Based Deployment
- High-Speed receivers: Use for deep routes
- High-Strength defenders: Use for physical coverage
- High-Hands players: Target for interception opportunities
- High-IQ players: Rely on for critical decisions
